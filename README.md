# Professional Portfolio

This is a personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Space Theme**: Animated background with floating tech icons and falling rockets.
- **Responsive Design**: Fully responsive layout for all devices.
- **Bento Grid**: Modern layout for the About section.
- **Theme Toggle**: Light and Dark mode support.

## Deployment
This project is ready to be hosted on GitHub Pages or Vercel.

### GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Select `main` branch and `/root` folder.
4. Save.
